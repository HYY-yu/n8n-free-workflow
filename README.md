## Instructions for Use

1. Install and run n8n. Refer to the official documentation: https://docs.n8n.io/getting-started/installation/

2. Download the workflow file from this repository.

3. Open the n8n editor, click "Create Workflow" in the top menu, click the upper right corner, and select "import from file..."

4. Select the file to import the workflow.

5. Configure credentials and parameters as needed.

6. Start the workflow and begin automating!
